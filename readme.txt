This game was made during the Neuro Game Jam 2019's edition and released under the MIT License.

Tools:
- Packager - custom c# .net solution - packs all assets using lzh algorithm
- Font Builds => GHL format => generates font texture and xml mapping information
- Android SDK
- Android NDK
- Inkscape
- Audicity/Reaper
- Sublime text

To build and run use the android folder batch files (build.bat, run.bat, clean.bat)
